Botium Toys Security Audit 🔐

This repository contains a security audit project completed as part of the Google Cybersecurity Professional Certificate.

📌 Project Overview

The goal of this project was to perform a security audit for a fictional company called Botium Toys and evaluate its security posture.

The audit includes:

Risk assessment
Security controls evaluation
Compliance analysis (PCI DSS, GDPR, SOC)
Security recommendations and remediation plan

Security documentation
Writing professional security reports
